# 🎯 G-Code Studio

**Генератор и визуализатор G-кода для фрезерных станков с ЧПУ**

## ✨ Возможности

### 🔧 Генератор G-кода

| Тип обработки | Описание |
|---------------|----------|
| **Зигзаг по X** | Фреза движется вдоль оси X с шагом по Y |
| **Зигзаг по Y** | Фреза движется вдоль оси Y с шагом по X |
| **От центра змейкой** | Спиральная обработка от центра или к центру |
| **Контурное** | Прямоугольные контуры от края к центру или наоборот |

### ⚙️ Настраиваемые параметры

- Размеры поля обработки (X, Y)
- Диаметр фрезы и шаг между проходами (stepover)
- Глубина обработки (Z start, Z end, Z step)
- Безопасная высота Z
- Подачи XY, Z и G0
- Скорость шпинделя
- Припуск (выход фрезы за границы поля)
- Направление фрезерования (попутное/встречное)
- Направление контурной обработки
- **Режим строгания** (подъём между проходами)
- Разбивка на файлы по количеству проходов

### 📊 Визуализатор

- **3 проекции:** XY, XZ, YZ
- **Анимация** с регулировкой скорости (1–50 кадров)
- **Пошаговое выполнение** (◀ Назад / Вперёд ▶)
- **Клик по строке кода** → переход к точке на графике
- **Подсветка** текущей строки и строки при наведении
- **Цветовое кодирование:** G0 (красный), G1 (синий), след фрезы (зелёный)
- **Оптимизированная отрисовка** через `LineCollection`

### ⏱️ Расчёт времени обработки

- Автоматический расчёт времени на основе траектории
- Учитывает скорость G0 (быстрые перемещения)
- Учитывает подачи XY, Z
- Выводится в лог при валидации и загрузке файла

### 🔄 Режим "Строгание"

- Фреза поднимается на безопасную высоту между проходами
- Все проходы идут в одном направлении
- Идеально для жёстких материалов (нержавейка, титан)
- Улучшает качество поверхности
