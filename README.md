**Генератор и визуализатор G-кода для фрезерных станков с ЧПУ**

## 📋 Описание

G-Code Studio — это открытое Python-приложение с графическим интерфейсом для:

- **Генерации G-кода** различными стратегиями фрезерования
- **Визуализации траекторий** инструмента в проекциях XY, XZ, YZ
- **Анимации процесса** обработки с отображением координат
- **Пошагового анализа** G-кода с подсветкой строк
- **Валидации параметров** перед генерацией

---

## ✨ Возможности

### 🔧 Генератор G-кода

| Тип обработки | Описание |
|---------------|----------|
| **Зигзаг по X** | Фреза движется вдоль оси X с шагом по Y |
| **Зигзаг по Y** | Фреза движется вдоль оси Y с шагом по X |
| **От центра змейкой** | Спиральная обработка от центра или к центру |
| **Контурное** | Прямоугольные контуры от края к центру или наоборот |

### ⚙️ Настраиваемые параметры

- Размеры поля обработки (X, Y)
- Диаметр фрезы и шаг между проходами (stepover)
- Глубина обработки (Z start, Z end, Z step)
- Безопасная высота Z
- Подачи XY и Z
- Скорость шпинделя
- Припуск (выход фрезы за границы поля)
- Направление фрезерования (попутное/встречное)
- Направление контурной обработки
- Разбивка на файлы по количеству проходов

### 📊 Визуализатор

- **3 проекции:** XY, XZ, YZ
- **Анимация** с регулировкой скорости (1–50 кадров)
- **Пошаговое выполнение** (◀ Назад / Вперёд ▶)
- **Клик по строке кода** → переход к точке на графике
- **Подсветка** текущей строки и строки при наведении
- **Цветовое кодирование:** G0 (красный), G1 (синий), след фрезы (зелёный)
- **Оптимизированная отрисовка** через `LineCollection`

### 📝 Дополнительные функции

- Поиск по тексту G-кода
- Копирование и сохранение файлов
- Кликабельные ссылки в логе
- Сохранение лога с автоматическим именем (дата + тип обработки)
- Заголовок с параметрами в каждом файле G-кода
- Имя файла содержит название метода обработки

## 📦 Установка

### Требования

- **Python 3.7+**
- **Операционная система:** Windows 10+, macOS 10.14+, Linux

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/Spruts80/CNC_Surface.git
cd gcode-studio
```

## Шаг 2: Установка зависимостей
```
pip install -r requirements.txt
```
Примечание для Linux: Tkinter может потребоваться установить отдельно:
```
sudo apt install python3-tk
```

Шаг 3: Запуск
```
python main.py
```

### Сборка EXE (Windows)

запуск установки pyinstaller в командной строке
```
pip install pyinstaller
```
потом запуск генератора EXE файла
```
build_gcode_studio.bat
```
или
```
Build.ps1
```
или
```
pyinstaller GCodeStudio.spec
```
в командной строке, готовый файл будет в папке dist/.

