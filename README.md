# G‑Code Studio

**G‑Code Studio** — это десктопное приложение для генерации, редактирования и визуализации G‑кода для фрезерных станков с ЧПУ. Оно позволяет создавать управляющие программы по различным стратегиям (зигзаг, спираль, контурная обработка), просматривать траекторию в 2D/3D, анимировать движение фрезы, редактировать код и сохранять настройки.

## Особенности

- **Генератор G-кода**  
  - Четыре стратегии: зигзаг по X/Y, спираль от центра, контурная обработка.  
  - Режим строгания (подъём между проходами).  
  - Настройка припуска, подач, оборотов, диаметра фрезы.  
  - Автоматическая разбивка на несколько файлов по количеству проходов.  

- **Визуализатор**  
  - Отображение траектории в проекциях XY, XZ, YZ и 3D.  
  - Анимация движения фрезы с отображением пройденного пути и следа обработки.  
  - Управление анимацией: Play/Pause/Stop, пошаговое перемещение, регулировка скорости.  
  - В 3D-режиме фреза отображается в виде полупрозрачного цилиндра.  

- **Редактор G-кода**  
  - Просмотр и редактирование кода внутри программы.  
  - Подсветка синтаксиса G-кода.  
  - Возможность применить изменения и перестроить траекторию.  

- **Окно просмотра файлов**  
  - Открытие любого G-кода из лога или через диалог.  
  - Поиск по тексту, подсветка найденного.  
  - Редактирование и сохранение с перезаписью исходного файла.  

- **Статистика траектории**  
  - Количество точек, общая длина пути, рабочий и холостой путь, максимальный шаг, минимальная/максимальная Z.  
  - Расчётное время обработки.  

- **Настройки**  
  - Все параметры сохраняются в файл `settings.json` рядом с приложением.  
  - Импорт/экспорт настроек через диалог.  
  - Поддержка светлой и тёмной темы.
